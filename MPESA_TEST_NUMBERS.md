# M-Pesa Sandbox Test Phone Numbers

## Available Test Phone Numbers

M-Pesa sandbox provides multiple test phone numbers you can use:

### Standard Test Numbers:
- `254708374149` - Most commonly used
- `254712345678` - Alternative test number
- `254700000000` - Another test option
- `254711111111` - Additional test number

### How to Use Different Numbers:

1. **In Payment Form**: Simply enter any of these numbers when making a payment
2. **Test PIN**: Use the test PIN from your Daraja portal (usually shown in test credentials)
3. **All numbers work**: Any of these numbers will work for sandbox testing

## Important Notes:

- **Test PIN**: You'll need the test PIN from your Daraja portal
- **No real money**: These are test numbers - no real money is involved
- **Same credentials**: All test numbers use the same shortcode (174379) and passkey
- **Sandbox only**: These only work in sandbox, not production

## Finding Test PIN:

1. Go to your Daraja portal
2. Navigate to "Lipa na M-Pesa Online" section
3. Look for "Test Credentials" or "Test PIN"
4. The test PIN is usually: `Safaricom123!!` or shown in the portal

## Using Multiple Numbers in Your App:

You can test with different numbers by:
- Entering different numbers in the payment form
- Creating test bookings with different customer accounts
- Using different phone numbers for each test payment
